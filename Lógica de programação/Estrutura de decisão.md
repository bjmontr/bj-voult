***If***
```javascript
	if(`teste lógico`){
		`comando`
	}if(`teste lógico`){
		`comando`
	}
```

***Else***
```javascript
	if(`teste lógico`){
		`comando`
	} else{
		`comando`
	}
```

***Else-if***
```javascript
	if(`teste lógico`){
		`comando`
	} else if (`teste lógico`){
		`comando`
	} else {
		`comando`
	}
```
