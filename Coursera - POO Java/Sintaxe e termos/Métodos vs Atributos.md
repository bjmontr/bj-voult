## Principais Diferenças:


| Aspecto   | Atributos                                  | Métodos |
| --------- | ------------------------------------------ | ------- |
| Definição | Representam o estado ou dados de um objeto |         |

## Atributos

#### Exemplos

##### Exemplo em uma classe

```java
public class Carro {
	int potencia;
	float velocidade;
	String cor;
}
```

##### Exemplo em um objeto

```java
civic.potencia = 150;
civic.velocidade = 140.5f;
civic.cor = "azul";
```
