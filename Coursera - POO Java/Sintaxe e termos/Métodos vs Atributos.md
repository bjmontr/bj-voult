## Atributos

### Definição

Os atributos são as características ou propriedades de uma classe ou objeto. Eles armazenam dados sobre o estado do objeto.

#### Exemplos

##### Exemplo em uma classe

```java
public class Carro {
	int potencia;
	float velocidade;
	String cor;
}
```

##### Exemplo em um objeto

```java
civic.potencia = 150;
civic.velocidade = 140.5f;
civic.cor = "azul";
```
